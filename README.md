# Tmux

## 参考文档

* [Tmux使用手册](http://louiszhai.github.io/2017/09/30/tmux/)

## 使用方法

* clone这个仓库；
* shell执行`auto.sh`：
  * install lastest tmux
  * link .tmux.conf file
* relaod config: `ctrl+b r`
* paste: `ctrl+b a`
* use predefine layout: `ctrl+b ctrl+a`
